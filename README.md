# Donut

Donut -- 使用  vue + vue-router 构建的web应用。

## Build setup

```bash
# install dependencies
npm install
```

```bash
# development
npm run dev
```

```bash
# build for production
npm run build
```

## 帮助
如果 npm install 比较慢，建议使用淘宝的镜像安装。
配置：`npm config set registry=https://registry.npm.taobao.org`

```bash
# use
cnpm install
```
